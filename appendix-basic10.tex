\chapter{BASIC 10 Command Reference}

\section{Commands}

\subsection{ABS (Absolute value function)}

                           ABS (expression)

The  ABSolute value function returns the unsigned value of the numeric
expression.

\begin{screenoutput}
                X = ABS(1)
\end{screenoutput}
              Result is X = 1
\begin{screenoutput}
                X = ABS(-1) 
\end{screenoutput}
            Result is X = 0
